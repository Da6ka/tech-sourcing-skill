Tech Sourcing — live demo

Paste a job description. This calls the real tech-sourcing-skill running on Claude — it builds a persona, runs live web searches across LinkedIn and adjacent platforms (GitHub, Stack Overflow, hh.ru, etc. depending on the role), and returns a scorecard plus outreach drafts. Takes 30–150 seconds since it's doing real searches, not a canned response.

Rate-limited to 5 runs/hour per visitor. No candidate data is stored server-side beyond the request/response. Source: github.com/Da6ka/tech-sourcing-skill
